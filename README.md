# Tic-tac-toe
Command line tic-tac-toe, one player vs smart A.I.
## Example Game
Run: `python tic-tac-toe.py`<br>
```
-------------
|   |   |   |
-------------
|   |   |   |
-------------
|   |   |   |
-------------
```
`Choose move e.g. 0 0:` 1 1<br>

```
-------------
|   |   |   |
-------------
|   | X |   |
-------------
|   |   |   |
-------------
```
AI plays
```
-------------
|   |   |   |
-------------
|   | X |   |
-------------
| O |   |   |
-------------
```

`Choose move e.g. 0 0:` 1 0<br>

```
-------------
|   | X |   |
-------------
|   | X |   |
-------------
| O |   |   |
-------------
```
AI plays
```
-------------
|   | X |   |
-------------
| O | X |   |
-------------
| O |   |   |
-------------
```

`Choose move e.g. 0 0:` 1 2<br>
```
-------------
|   | X |   |
-------------
| O | X |   |
-------------
| O | X |   |
-------------
```
